# terraform-azure-centos-vm
Deploying CentOS VM in Azure using Terraform

To change the CentOS version, update the SKU in the **linux-vm-main.tf** file using the values in the **centos-variables.tf**

Versions included:

* CentOS 7.x
* CentOS 8.x
